# live_compile_hitex

update `.tex` into `.hnt` by hilatex, whenever changes happen

Usage: live_compile_hitex [DIR]

Arguments:
  [DIR]  directory to watch [default: .]

Options:
  -h, --help     Print help
  -V, --version  Print version
